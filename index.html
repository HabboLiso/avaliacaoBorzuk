Curso de Desenvolvimento de Sistemas

O curso de Desenvolvimento de Sistemas da SETEC prepara os alunos para os desafios do mercado de tecnologia, fornecendo conhecimentos sólidos em programação, banco de dados, desenvolvimento web e mobile.

Nossa abordagem prática e inovadora permite que os estudantes desenvolvam projetos reais desde o primeiro semestre, criando um portfólio consistente ao longo do curso.

Temas em Inteligência Artificial

Machine Learning

Algoritmos que permitem aos sistemas aprenderem e melhorarem com experiência sem serem explicitamente programados.

  • Aprendizado supervisionado
  • Aprendizado não supervisionado
  • Redes neurais artificiais

Robótica Inteligente

Integração de IA em sistemas robóticos para criação de máquinas autônomas capazes de interagir com o ambiente.

  • Visão computacional
  • Automação industrial
  • Robôs de serviço

Processamento de Linguagem Natural

Tecnologias que permitem aos computadores entender, interpretar e gerar linguagem humana de forma natural.

  • Chatbots e assistentes virtuais
  • Análise de sentimentos
  • Tradução automática

Ciência de Dados

Extração de insights valiosos a partir de grandes volumes de dados usando técnicas estatísticas e de IA.

  • Mineração de dados
  • Big Data analytics
  • Visualização de dados
